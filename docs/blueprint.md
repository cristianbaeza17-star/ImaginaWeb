# **App Name**: ImaginaWeb

## Core Features:

- Prompt Generator: Generates creative prompts to spark ideas, using AI to ensure variety and novelty.
- Random Scenarios: Presents users with random, unexpected scenarios to challenge their thinking.
- Idea Association Tool: Provides a tool that assists users in associating seemingly unrelated concepts, fostering innovative connections. An LLM is used as a tool to judge whether a connection is meaningful, or trite.
- Creative Exercises: Offers a selection of structured exercises designed to improve creative problem-solving.
- Community Sharing: Allows users to share their creative outcomes and insights with a supportive community.

## Style Guidelines:

- Primary color: Vivid purple (#9D4EDD) to inspire imagination and creativity.
- Background color: Light lavender (#E9D5FF) for a soft, inviting atmosphere.
- Accent color: Electric blue (#577590) to highlight interactive elements and CTAs.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text to combine modernity with readability.
- Use abstract and symbolic icons to represent different creative activities.
- Maintain a clean, uncluttered layout with ample white space to promote focus and clarity.
- Incorporate subtle, playful animations to add a touch of whimsy and delight.
